Premiers pas

Que fait Pod2Swim ?

Il charge des épisodes de podcast sur un Shokz OpenSwim / OpenSwim Pro — ou n'importe quel casque lecteur MP3 qui se monte comme un stockage USB — pour que tu puisses écouter pendant que tu nages, là où le Bluetooth ne passe pas. Abonne-toi, mets des épisodes en file d'attente, et l'app les télécharge puis les copie directement sur le casque dans l'ordre de lecture.

Comment mettre des épisodes sur mon casque ?

Abonne-toi à un podcast, ajoute les épisodes que tu veux à la File d'attente, et attends qu'ils finissent de se télécharger (chaque ligne indique son état). Branche le casque sur ton iPhone avec un adaptateur USB‑C ou Lightning, ouvre l'onglet File d'attente et touche Envoyer vers le casque. La première fois, sélectionne le dossier du casque dans le sélecteur Fichiers — l'app le mémorise ensuite.

De quoi ai-je besoin ?

Un iPhone sous iOS 17 ou version ultérieure, une connexion internet pour les téléchargements, et un adaptateur USB‑C ou Lightning pour que le casque apparaisse comme un stockage lisible dans le sélecteur Fichiers.

Utiliser l'app

Tout remplacer ou Ajouter à la fin — quelle différence ?

Tout remplacer supprime les fichiers audio déjà présents sur le casque et écrit une nouvelle playlist à partir de ta file d'attente. Ajouter à la fin conserve ce qui est là et ajoute ta file d'attente à la suite. On te demande à chaque fois, avec une option « ne plus me demander » qui mémorise ton choix.

Pourquoi mes épisodes ne se lisent-ils pas dans le bon ordre ?

L'OpenSwim Pro lit les fichiers dans l'ordre où ils ont été transmis, ce qui correspond à leurs horodatages de fichier. Pod2Swim horodate chaque fichier exporté dans l'ordre croissant selon la position dans la file d'attente, donc la position 1 se lit en premier. Si l'ordre semble toujours erroné, réexporte avec Tout remplacer pour que le casque reparte d'un jeu propre.

Puis-je accélérer la lecture ?

Oui. Chaque podcast a un réglage de vitesse de 0,75× à 2,5×. Lorsqu'un épisode se télécharge, il est traité en un fichier accéléré avec la hauteur de son préservée, et c'est lui qui est exporté. Définis la valeur par défaut pour les nouveaux abonnements dans les Réglages, ou modifie-la par podcast dans la Bibliothèque.

Le casque n'est pas détecté quand je touche Envoyer ou Analyser.

Assure-toi que le casque est branché et allumé, et qu'il apparaît sous « Emplacements » dans l'app Fichiers. Si le dossier enregistré est inaccessible, l'app bascule vers le sélecteur Fichiers — re‑sélectionne le dossier du casque à cet endroit. Certains adaptateurs doivent être bien enfoncés ; essaie de rebrancher le câble.

Comment libérer de l'espace de stockage ?

Retirer un épisode de la file d'attente supprime ses fichiers téléchargés. Les Réglages affichent le stockage total téléchargé avec un bouton Supprimer tous les téléchargements, ainsi qu'un délai de suppression automatique qui efface les fichiers exportés un certain nombre de jours après l'export (7 jours par défaut).

Confidentialité

Pod2Swim ne collecte aucune donnée personnelle — pas de compte, pas d'analytique, pas de publicité. Tout reste sur ton appareil. Lis la politique de confidentialité complète.
